† Phase-gated write strobes (all on PLD 3, the CLK device): the cell shows the write condition; the physical pin is !(cond · /CLK) — asserted only in the settled second half of the cycle (WRPH = /CLK; quarter-cycle provision unfitted). /DSTK_CS is not a PLD output: it is /DSP_EN wired directly; /RSTK_CS is explicit because R@ reads at RSP−1 with the counter disabled. There is no VEC_EN: the '139 half that decodes PCSEL into the three NEXTPC enables yields the vector enable as Y3 (PCSEL = 11); the vector lines are bit 1 = NMI_PEND, bit 0 = NMI_PENDB — NMI → 0x0002, IRQ/BRK → 0x0001. B is not decoded: B = IR[10] wired to the BP-lane data-in (1 on BRK 0x0400, 0 on INT 0x0300). The I flag is stored inverted as IBAR (1 = unmasked); ← marks what the register loads (←s̄ = complement of the saved lane bit on RTI); the front-panel I LED is IBAR inverted. NMI entry overrides CLK_RUN and wakes a HALTed machine. /ALU_CN on ADC shows C: the pin is the complement of the C flag (inject when C clear).

Instructions

NOP
No operation; PC advances. Word 0x0000.
HALT
Stop the clock; NMI wakes.
RET
Pop {PC, BP}; flags untouched.
BRK
Software trap: push {PC+1, N,Z,C} + {BP, I, B=1}; PC ← 0x0001; I ← 1.
RTI
Pop both lanes; PC, BP, N/Z/C, I all restored.
CLI / SEI
I ← 0 / I ← 1 (I = IR[8] on the shared cube).
ADD / ADC / SUB
( a b — result ), pop. SUB = b−a; C = no borrow. ADC reads C.
AND / OR / XOR
( a b — result ), pop. Logic ops preserve C.
NOT
( a — ~a ). Preserves C.
TST
Flags of TOS; TOS held; preserves C.
CMP
( a b — a b ) flags of TOS−NOS; C = TOS ≥ NOS. No stack activity.
SHL / SHR
'194 shift, zero fill; C ← bit shifted out.
ASR
'194 right shift, sign fill; C ← old bit 0.
ROL / ROR
'194 plain rotates (wrap); C ← bit rotated out.
DUP / OVER / TUCK
Pushes: ( a — a a ), ( a b — a b a ), ( a b — b a b ). TUCK writes old TOS to the stack (SDIN_SEL).
SWAP / DROP / NIP
( a b — b a ), ( a — ), ( a b — b ).
>R / R> / R@
Move 8-bit data to / from / copy the return-stack PC lane. R@ reads at RSP−1 without counting.
PUSH #
( — imm ) push IR[7:0].
FETCH
( addr — value ) D-mem read, addr = TOS (low 256 cells).
IN / IN #
Port read: stack form (port = TOS, ΔDSP 0) / immediate form (+1).
OUT #
( d — ) write TOS to port IR[7:0]. No stack-form OUT.
ENTER
BP ← BP + k via the BP adder.
LOCAL@ / LOCAL!
Push / pop D-mem[BP + signed offset].
BRA … BPL
PC ← PC+1+offset; taken-ness lives entirely in BSEL, PCSEL stays 00.
JUMP / CALL
PC ← IR[12:0]; CALL also pushes {PC+1, BP}.
LOAD / STORE
Push D-mem[IR[12:0]] / pop → D-mem[IR[12:0]].
IRQ / NMI entry
Inserted cycle: BRK_FORCE jams INT (0x0300, jam-only CTL member 3) onto the decode inputs; pushes raw PC (RDIN = 01) so the displaced instruction re-executes; B = IR[10] = 0.
reserved
ALU nibbles A/D, unassigned members, families 110/111: decode inert — no stack, RAM, I/O, flag or BP activity.

Control signals

ALU_S3–S0
'181 function select. Shared codes: SUB/XOR/CMP = 0110, ADD/ADC = 1001, TST = 1111, AND = 1011, OR = 1110, NOT = 0000.
ALU_M
'181 mode: 1 = logic, 0 = arithmetic.
/ALU_CN
'181 carry-in pin (active-low inject): 1 = no carry, 0 = +1 (SUB/CMP); on ADC the pin is the complement of C.
C_SRC
C-flag source: 0 = '181 Cn+4 chain, 1 = '194 shifted-out bit.
TOS_M1 M0
Raw '194 mode pins: 00 hold, 01 shift right, 10 shift left, 11 parallel load.
TOS_SRC2–0
TOS load source: 000 ALU, 001 NOS, 010 IR[7:0], 011 D-mem, 100 I/O, 101 RTOS[7:0].
NOS_LD / NOS_SRC
NOS latch enable; source 0 = old TOS (push spill / SWAP), 1 = stack[DSP−1] read (pops).
SDIN_SEL
Stack data-in '257: 0 = NOS (normal push spill), 1 = TOS (TUCK).
NZ_WE / C_WE
Per-flag latch enables. NZ on every live ALU sub-op; C only on ADD/ADC/SUB/CMP + the five shifts. Both assert on RTI (restore path via the '157).
/DMEM_CS /WE
D-memory select (PLD 2) / write strobe (PLD 3, phase-gated †).
/DSP_EN DSP_UD
Data-stack '191 /CTEN and D//U: 0 = up (push), 1 = down (pop). /DSP_EN doubles as /DSTK_CS.
/RSP_EN RSP_UD
Return-stack '191 /CTEN and D//U, 0 = up. RSP_UD also steers the '157 address mux: up → RSP (write), down → RSP−1 (read). R@: UD = 1 with /RSP_EN = 1.
/RSTK_CS
Return-stack SRAM select: push/pop/R@ cycles (explicit output; R@ breaks the /RSP_EN derivation).
/DSTK_WE /RSTK_WE
Stack write strobes, phase-gated †.
BP_LD / BP_SRC
BP register load: ENTER, RET, RTI only. Source 0 = BP adder (ENTER), 1 = saved-BP lane (RET, RTI).
DADDR_S1–0
D-address mux: 00 IR[12:0], 01 TOS zero-ext, 10 BP + sign-ext IR[7:0].
BSEL
PC-adder B input: 0 = +1, 1 = sign-extended IR[7:0] (taken branch). On PLD 4 with the flags.
CLK_RUN
0 stops the clock (HALT). NMI_PEND overrides.
/IO_RD /IO_WR
Port read (PLD 4, level) / write (PLD 3, phase-gated †).
IOADDR_SEL
Port-number mux: 0 = IR[7:0], 1 = TOS.
RDIN_SEL1–0
PC-lane data-in: 00 PC+1 (CALL, BRK), 01 raw PC (INT entry), 10 TOS zero-ext (>R).
PCSEL1–0
NEXTPC source: 00 adder, 01 IR[12:0], 10 RTOS, 11 vector ('139 Y3 = vector enable).
BRK_FORCE
= NMI_PEND # IRQ_PEND & IBAR. Selects the '257 pair that jams INT onto the decode side of the IR.
IBAR
The I flag, stored inverted (1 = IRQs enabled). AR clears to 0 = masked on reset; SEI/BRK/INT clear it, CLI sets it, RTI loads !ILANE.
IRQ_PEND
Registered level sample of the IRQ line.
NMI_SYNC / NMI_PEND
NMI synchronizer and true-edge pend: NMI_PEND.d = NMI & !NMI_SYNC — one-cycle pend, consumed by the entry cycle, no retrigger while held.
NMI_PENDB
Registered complement of NMI_PEND; vector address bit 0.
