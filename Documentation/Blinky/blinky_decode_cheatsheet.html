† Phase-gated write strobes: the cell shows the write condition for the cycle; the physical pin is !(cond · /CLK) — the strobe asserts only in the settled second half of the cycle (WRPH = /CLK; quarter-cycle provision unfitted). /DSTK_CS is not a PLD output: it is /DSP_EN directly. NMI entry overrides CLK_RUN, so it wakes a HALTed machine. IRQ_PEND / NMI_PEND / I are PLD 5 registered macrocells — state, not decode; ← marks the value the cycle loads (←s = restored from the saved BP-lane bits on RTI).

Instructions

NOP
No operation; PC advances. Word 0x0000.
HALT
Stop the clock; NMI wakes.
RET
Pop {PC, BP}; flags untouched.
BRK
Software trap: push {PC+1, N,Z,C} + {BP, I, B=1}; PC ← 0x0001; I ← 1.
RTI
Pop both lanes; PC, BP, N/Z/C, I all restored.
CLI / SEI
I ← 0 / I ← 1 (I = IR[8] on the shared cube).
ADD / ADC / SUB
( a b — result ), pop. SUB = b−a; C = no borrow. ADC reads C.
AND / OR / XOR
( a b — result ), pop. Logic ops preserve C.
NOT
( a — ~a ). Preserves C.
TST
Flags of TOS; TOS held; preserves C.
CMP
( a b — a b ) flags of TOS−NOS; C = TOS ≥ NOS. No stack activity.
SHL / SHR
'194 shift, zero fill; C ← bit shifted out.
ASR
'194 right shift, sign fill; C ← old bit 0.
ROL / ROR
'194 plain rotates (wrap); C ← bit rotated out.
DUP / OVER / TUCK
Pushes: ( a — a a ), ( a b — a b a ), ( a b — b a b ). TUCK writes old TOS to the stack (SDIN_SEL).
SWAP / DROP / NIP
( a b — b a ), ( a — ), ( a b — b ).
>R / R> / R@
Move 8-bit data to / from / copy the return-stack PC lane. R@ reads at RSP−1 without counting.
PUSH #
( — imm ) push IR[7:0].
FETCH
( addr — value ) D-mem read, addr = TOS (low 256 cells).
IN / IN #
Port read: stack form (port = TOS, ΔDSP 0) / immediate form (+1).
OUT #
( d — ) write TOS to port IR[7:0]. No stack-form OUT.
ENTER
BP ← BP + k via the BP adder.
LOCAL@ / LOCAL!
Push / pop D-mem[BP + signed offset].
BRA … BPL
PC ← PC+1+offset; taken-ness lives entirely in BSEL, PCSEL stays 00.
JUMP / CALL
PC ← IR[12:0]; CALL also pushes {PC+1, BP}.
LOAD / STORE
Push D-mem[IR[12:0]] / pop → D-mem[IR[12:0]].
IRQ / NMI entry
Inserted cycle: BRK_FORCE jams 0x0400 onto decode; pushes raw PC (RDIN = 01) so the displaced instruction re-executes; B = 0 in the lane.
reserved
ALU nibbles A/D, unassigned members, families 110/111: decode inert — no stack, RAM, I/O, flag or BP activity.

Control signals

ALU_S3–S0
'181 function select. Shared codes: SUB/XOR/CMP = 0110, ADD/ADC = 1001, TST = 1111, AND = 1011, OR = 1110, NOT = 0000.
ALU_M
'181 mode: 1 = logic, 0 = arithmetic.
ALU_Cn
'181 carry-in pin (active-low carry): 1 = no carry, 0 = +1 (SUB/CMP), C̄ on ADC.
C_SRC
C-flag source: 0 = '181 Cn+4 chain, 1 = '194 shifted-out bit.
TOS_M1 M0
Raw '194 mode pins: 00 hold, 01 shift right, 10 shift left, 11 parallel load.
TOS_SRC2–0
TOS load source: 000 ALU, 001 NOS, 010 IR[7:0], 011 D-mem, 100 I/O, 101 RTOS[7:0].
NOS_LD
NOS latch enable.
NOS_SRC
NOS source: 0 = old TOS, 1 = stack[DSP−1] read.
SDIN_SEL
Stack data-in '257: 0 = NOS (normal push spill), 1 = TOS (TUCK).
NZ_WE / C_WE
Per-flag latch enables. NZ on every live ALU sub-op; C only on ADD/ADC/SUB/CMP + the five shifts. Both assert on RTI (restore path via the '157).
/DMEM_CS /WE
D-memory select / write strobe (WE phase-gated †).
/DSP_EN
Data-stack '191 /CTEN. Doubles as /DSTK_CS.
DSP_UD
'191 D//U: 0 = up (push), 1 = down (pop).
/RSP_EN
Return-stack '191 /CTEN. Deasserted on R@ (read without count).
RSP_UD
'191 D//U, 0 = up. Also steers the '157 address mux: up → RSP (write), down → RSP−1 (read).
RDIN_SEL1–0
PC-lane data-in: 00 PC+1 (CALL, BRK), 01 raw PC (hardware entry), 10 TOS zero-ext (>R).
/RSTK_CS
Return-stack SRAM select: push/pop/R@ cycles.
/DSTK_WE /RSTK_WE
Stack write strobes, phase-gated †.
BP_LD
BP register load: ENTER, RET, RTI only — never R>/R@.
BP_SRC
0 = BP adder (ENTER), 1 = saved-BP lane (RET, RTI).
DADDR_S1–0
D-address mux: 00 IR[12:0], 01 TOS zero-ext, 10 BP + sign-ext IR[7:0].
/IO_RD /IO_WR
Port read / write strobes (WR phase-gated †).
IOADDR_SEL
Port-number mux: 0 = IR[7:0], 1 = TOS.
CLK_RUN
0 stops the clock (HALT). NMI_PEND overrides.
PCSEL1–0
NEXTPC source: 00 adder, 01 IR[12:0], 10 RTOS, 11 vector.
BSEL
PC-adder B input: 0 = +1, 1 = sign-extended IR[7:0] (taken branch).
VEC_EN
Vector driver enable = (PCSEL = 11), emitted directly.
BRK_FORCE
'257 pair jams 0x0400 onto the decode-input side of the IR.
IRQ_PEND NMI_PEND I
PLD 5 registered macrocells: level-sampled IRQ, edge-latched NMI, interrupt mask. I = 1 on reset (async-reset term).
