Active-low signals (marked with a leading /; asserted = 0 = LED off): /DMEM_CS, /DMEM_WE, /DSP_EN, /RSP_EN, /ALU_CN, /IO_RD, /IO_WR. For these, a lit LED means de-asserted; dark means the signal is doing its job.

Phase-gated strobes /DMEM_WE and /IO_WR are shown asserted (0) with the write-phase window WRPH = 1. On a static bench with WRPH not driven high they stay 1 (LED on). Drive WRPH high to see STORE pull /DMEM_WE low and OUT pull /IO_WR low.

Flag cells (yellow) read the GAL4 flag inputs: BEQ→PCSEL0 = Z, BCS→PCSEL0 = C, BMI→PCSEL0 = N, and ADC→ALU_CN = C̄ (complement of C). Set the matching flag input to predict that LED.

GAL4 pins 13 / 12 are the C and Z flag inputs (blue/grey cells), not decode outputs — those two LEDs mirror your flag switches. The N flag is on pin 11, off the 19–12 bar.

GAL3 pin 12 is an unused spare output.

Low-nibble literals: for MEMORY and CONTROL-FLOW ops the low nibble carries data, shown here as 0000. The GAL1 select group (ALU_S0–S3, ALU_M) is un-qualified by opcode, so on the bench those five LEDs track whatever you set in the blue switches — they are don't-care for these instructions. Every other column is opcode-determined and holds as shown.

Instructions

NOP
No operation; PC advances.
HALT
Stop the clock; freeze the panel.
RET
Return: PC <- return-stack top.
DUP
Duplicate TOS onto the stack.
DROP
Discard TOS.
SWAP
Exchange TOS and NOS.
>R
Push TOS onto the return stack.
R>
Pop return stack onto TOS.
ADD
TOS = NOS + TOS.
ADC
TOS = NOS + TOS + C.
SUB
TOS = NOS - TOS.
XOR
TOS = NOS ^ TOS.
NOT
TOS = ~TOS.
TST
Set N/Z from TOS; TOS unchanged.
SHL
Shift TOS left, 0 in, MSB -> C.
SHR
Shift TOS right, 0 in, LSB -> C.
AND
TOS = NOS & TOS.
CMP
Flags of NOS - TOS; stack unchanged.
ASR
Arithmetic shift right (sign fill).
OR
TOS = NOS | TOS.
ROL
Rotate TOS left through C.
ROR
Rotate TOS right through C.
PUSH #
Push literal (low nibble) onto stack.
LOAD
TOS <- D-memory[addr].
STORE
D-memory[addr] <- TOS (pop).
IN
TOS <- input port (port = TOS).
OUT #
Output port[#] <- TOS (pop).
BRANCH
PC <- literal (unconditional).
BEQ
PC <- literal if Z set.
BCS
PC <- literal if C set.
BMI
PC <- literal if N set.
CALL
Push PC+1, PC <- literal.
reserved
ALU nibbles A, D and opcodes C–F are undefined and decode inert (no stack, RAM, I/O or flag activity).

Control signals

ALU_S0–S3
'181 function-select lines (4 bits).
ALU_M
'181 mode: high = logic, low = arithmetic.
C_SRC
Carry source mux: 1 = bit shifted out of the '194, 0 = '181 Cn+4.
TOS_M1 TOS_M0
'194 mode pins S1/S0: 00 hold, 01 shift-right, 10 shift-left, 11 parallel-load.
TOS_SRC0–2
TOS source mux: 000 NOS, 001 ALU, 010 LIT, 011 DMEM, 100 INP, 101 RTOS.
NOS_LD
NOS register latch enable.
NOS_SRC
NOS source: 1 = old TOS, 0 = data-stack read.
NZ_WE
N and Z flag latch enable.
/DMEM_CS
D-memory 2114 chip select (active low).
/DMEM_WE
D-memory write strobe (active low, WRPH-gated).
/DSP_EN
Data-stack '169 count enable (active low).
DSP_UD
Data-stack direction: 1 = up (push), 0 = down (pop).
/RSP_EN
Return-stack '169 count enable (active low).
RSP_UD
Return-stack direction: 1 = up, 0 = down.
RDIN_SEL
Return data-in mux: 0 = TOS (>R), 1 = PC+1 (CALL).
CLK_RUN
1 = clock runs, 0 = stopped (HALT only).
C_WE
Carry flag latch enable.
PCSEL0 PCSEL1
Next-PC mux: 00 PC+1, 01 IR literal, 10 RTOS.
/ALU_CN
'181 carry-in pin (active-low assert: 0 injects +1).
/IO_RD
Input-port read strobe (active low).
/IO_WR
Output-port write strobe (active low, WRPH-gated).
IOADDR_SEL
I/O address mux: 0 = TOS, 1 = IR literal.
